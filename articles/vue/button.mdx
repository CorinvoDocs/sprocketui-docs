## Usage

The Button component for Vue provides a flexible, accessible button with support for multiple variants and states.

### Basic Example

```vue
<script setup>
import { Button } from '@sprocket-ui/vue-button';
</script>

<template>
  <Button @press="handleClick">
    Click me
  </Button>
</template>
```

### With Variants

```vue
<script setup>
import { Button } from '@sprocket-ui/vue-button';
</script>

<template>
  <div class="flex gap-4">
    <Button variant="primary">Primary</Button>
    <Button variant="secondary">Secondary</Button>
    <Button variant="outline">Outline</Button>
  </div>
</template>
```
