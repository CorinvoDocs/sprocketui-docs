## Usage

The Label component provides an accessible way to associate descriptive text with form controls.

### Basic Example

```tsx
import { Label } from '@sprocket-ui/label';

<Label htmlFor="username">Username</Label>
<input id="username" type="text" />

With Nested Control

<Label>
  <span>Remember me</span>
  <input type="checkbox" />
</Label>

Required Field

<Label htmlFor="email">
  Email
  <span className="text-red-500 ml-1">*</span>
</Label>
<input id="email" type="email" required />

With Description

<div className="flex flex-col gap-1">
  <Label htmlFor="password">Password</Label>
  <input id="password" type="password" />
  <span className="text-sm text-gray-500">
    Must be at least 8 characters
  </span>
</div>

Disabled State

<Label htmlFor="disabled-input" className="opacity-50 cursor-not-allowed">
  Disabled field
</Label>
<input id="disabled-input" type="text" disabled />

Accessibility

The Label component follows WAI-ARIA best practices:

- Uses semantic <label> element for proper form association
- Clicking the label focuses the associated form control
- Text selection is prevented on double-click to avoid accidental selection
- Screen readers properly announce the label when the form control is focused
- Supports both htmlFor association and nested control patterns

Styling

The Label component is unstyled by default and supports full customization through the className prop. Common styling patterns
include:

{/* Basic styling */}
<Label className="text-sm font-medium text-gray-700">
  Username
</Label>

{/* Required field indicator */}
<Label className="text-sm font-medium text-gray-700 after:content-['*'] after:ml-0.5 after:text-red-500">
  Email
</Label>

{/* Disabled appearance */}
<Label className="text-sm text-gray-400 cursor-not-allowed">
  Disabled field
</Label>
